Uppidi Upload

Uppidi Upload is an open source, cross-platform application for uploading media and files to public and private providers. Upload images, videos, and other files from your phone, tablet, or desktop and share the URL anywhere, including IRC (backed by a public provider of your choice for the file itself). Built-in image editor lets you crop, paint, add text, apply filters, emoji, blur, and pixelate before uploading or saving locally.

Features

  • 19 hosting providers — Catbox, FileDitch, Frisk, Litterbox, Uguu.se, TmpFile.link, FreeImage.host, TempSH, GoFile, Filebin.net, Filester.me, Storage.to, Bzzhr.to, HttpBin.org (test), plus your own Telegram Bot, Zulip, CustomUguu, Matterbridge, and Local (Image Editor)
  • Your own infrastructure — configure and authenticate with private servers: Telegram Bot, Zulip, self-hosted Uguu, and Matterbridge for IRC/Discord/Matrix/Slack relay
  • Built-in image editor — a full editing tab with crop, paint, text, filters, emoji, blur, and pixelate on a checkered canvas. Save locally or upload to any provider. Also functions as a standalone image editor
  • Desktop-first experience — drag files from your file manager, paste images from clipboard, real hover overlay feedback
  • Message templates — set a global template in Settings ({url}, {filename}, {filesize}) and edit per upload
  • History → IRC pipeline — re-share any past upload to a Matterbridge gateway from the history screen
  • Streaming uploads — handles large files without consuming excessive memory
  • My Providers dialog — manage custom instances and credentials for private services
  • Provider health checks — test connectivity and enable/disable providers
  • Settings backup — structured export and import of all app settings
  • Dark & light theme — system-following theme with customizable seed colors
  • Multi-language — English, Turkish, Italian, Esperanto, and Klingon (we do not discriminate against the Alien or the Woke)
  • Cross-platform — Android, Linux (tar.gz, AppImage, Flatpak), and Windows
